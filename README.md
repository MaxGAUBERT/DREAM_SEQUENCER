# DREAM_SEQUENCER

DREAM_SEQUENCER is a modular music production and sequencing application designed for intuitive pattern-based composition and live arrangement. It combines classic step sequencing workflows with modern editing tools to provide fluid control over rhythm, melody, and sound design.

---

## Core Concepts

DREAM_SEQUENCER is built around two main playback modes:

* **Pattern Mode** ‚Äì Loop-based playback of individual patterns for precise sound design and experimentation.
* **Song Mode** ‚Äì Linear arrangement by chaining multiple patterns inside the Playlist for full track composition.

---

## Workflow Overview

1. Create drum and melodic content using DrumRack and PianoRoll
2. Apply audio processing via FXChain and adjust channel volumes
3. Organize and switch patterns through PatternSelector
4. Arrange patterns in the Playlist
5. Play the full composition in Song Mode

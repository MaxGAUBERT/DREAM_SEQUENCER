# DREAM-SEQUENCER

# _Description_
## **Dream Sequenceur** est un séquenceur audio programmé en React JS, avec les librairies Material UI et Tone JS.
## Parmis les fonctionnalités, on retrouve:
### 1. Un **Channel rack** qui permet de charger des fichier .wav
### 2. Un **Sound browser** qui permet de charger un dossier de fichiers .wav
### 3. Un **Menu principal** qui gère des sous menu (comme l'import / export de projets ou les bascules d'affichages)
### 4. Un **PianoRoll** qui gère la création de mélodies grâce à un système de grille rythmique dont les dimensions sont personnalisables
### 5. Une **Playlist** permettant de placer des patterns dans une grille (elle aussi modifiable), et ainsi de créer des sons entiers
### 6. Un **Pattern Manager** qui permet d'ajouter / supprimer ou encore dupliquer des patterns
